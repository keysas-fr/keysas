Network gateway (keysas-core)ÔÉÅ

By installing Keysas-core alone, you can create an upstream gateway between two separate networks in order to filter files transmitted from the lower network to the upper one. To do so, use Makefile target make install-core. It will prevent the installation of binaries related to USB and webservices. See below to learn about the administration of Keysas-core and how to use it.
